# MirrorChat: AI一括質問＆Obsidian保存ツール

1つのプロンプトを ChatGPT / Claude / Gemini / Grok に順次送信し、回答を Obsidian に Markdown として保存する Chrome 拡張です。

## インストール

1. このリポジトリをクローンまたはダウンロードする
2. Chrome で `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `ai-prompt-broadcaster` フォルダを選択する

## 前提条件

### Obsidian 側

1. Obsidian に **Local REST API** プラグインをインストール・有効化する
2. プラグイン設定で API トークンとポート番号を確認する（HTTPS デフォルト: 27124）
3. CORS を許可する設定を行う（必要に応じて）

### 各 AI サービス

- ChatGPT, Claude, Gemini, Grok のいずれかにログイン済みであること
- 各サイトを開いた状態でなくても、拡張がタブを開いて自動操作します

## 使い方

1. 拡張アイコンをクリックしてポップアップを開く
2. 質問を入力して「送信」をクリック
3. バックグラウンドで 4 つの AI サイトが順次開き、プロンプトが送信される
4. 回答が取得されると Obsidian の Vault 内に保存される

### 保存先フォルダ構成

```
200-AI Research/
└── YYYY-MM-DD_質問の先頭20文字/
    ├── question.md    # 元の質問
    ├── ChatGPT.md
    ├── Claude.md
    ├── Gemini.md
    ├── Grok.md
    └── Summary.md    # 全回答をまとめたファイル
```

## 設定

拡張アイコン右クリック → 「オプション」、またはポップアップ内の「設定」リンクから設定画面を開けます。

- **Obsidian 連携**: ベース URL、API トークン、保存ルートパス
- **各 AI の DOM セレクタ**: 入力欄・送信ボタン・回答コンテナのセレクタ（UI 変更で動かなくなった場合に調整）

## 既知の制限

- 各 AI サービスの UI（DOM 構造）は頻繁に変更されます。動かなくなった場合は Options 画面でセレクタを調整してください
- Grok は OSS の参考実装が少なく、セレクタは実機で確認しながらチューニングが必要な場合があります
- Obsidian が起動していない、または Local REST API が無効な場合は保存に失敗します。失敗したデータはローカルに一時保存され、後から「再送信」できます
