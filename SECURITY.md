## 脆弱性の報告

セキュリティ上の脆弱性を発見した場合は、**GitHub の Issue を公開せず**、以下のいずれかの方法で非公開に報告してください。

- **GitHub Security Advisories**: リポジトリの「Security」タブ → 「Report a vulnerability」
- **メール**: リポジトリのメンテナに直接連絡

報告いただいた内容は速やかに確認し、必要に応じて対応します。

## 報告に含めていただきたい情報

- 脆弱性の種類と影響範囲
- 再現手順
- 想定される対策（あれば）

## 対応方針

- 重大な脆弱性: 48時間以内に初回対応
- その他: 可能な限り早く対応
