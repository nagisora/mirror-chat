# 翻訳運用ガイド

[English](TRANSLATIONS.md) | [日本語](TRANSLATIONS_ja.md)

このリポジトリでは、当面のドキュメント対応言語を英語と日本語に限定します。

## 対象範囲

- 言語 suffix のない英語ファイルを canonical とします。
- 日本語ファイルは `_ja.md` suffix を付け、対応する英語ファイルの内容を追従します。
- 内部向け調査メモは、ユーザーやコントリビューター導線に入るまでは日本語のみでも構いません。

## 更新ルール

1. まず英語の canonical 文書を更新します。
2. 可能な限り同じ変更で日本語版も更新します。
3. 同時更新できない場合は、Pull Request に未翻訳または未追従のファイルを明記します。

## ファイル命名

- `README.md` と `README_ja.md`
- `CONTRIBUTING.md` と `CONTRIBUTING_ja.md`
- `docs/NAME.md` と `docs/NAME_ja.md`

## レビュー観点

- リンクが正しい言語版を指しているか確認します。
- 強い理由がない限り、英語版と日本語版でセクション順序をそろえます。
- 機械翻訳の直訳より、技術的に正確な表現を優先します。

## 当面やらないこと

- 英語と日本語以外の追加言語対応
- `docs/ja/` のような言語別ディレクトリ構成
